一、集合是什么？
	集合论可以看成是逻辑的几何化。集合是最简单的空间。
	集合是指具有某种特定性质的具体的或抽象的对象汇总而成的集体。其中,构成集合的这些对象则称为该集合的元素
二、集合的表示
	通常用大写字母A B C ...表示集合
	小写字母a b c ...表示集合的元素(注意,一个集合也可以作为另一个集合的元素)
	常用的集合有N Z Q R
三、集合和元素的关系
	1.属于关系：如果元素a是集合A的元素,就说a属于A,记作a∈A
	2.不属于关系：反之,a不是集合A的元素,就说a不属于A,记作a∉A
四、集合的表示方法
	1.枚举法
		P = { a,b,c,d ... }
	2.叙述法
		P = { x|x=P(x) }
	3.文氏图法
五、集合基数
	集合A中元素的个数称为集合A的基数,记作|A|
	一个集合的基数是有限的,称这个集合为有限集；反之,一个集合的基数是无限的,称之为无限集
