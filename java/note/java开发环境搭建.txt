一、JDK是什么？
	JDK 是Java开发工具包 (Java Development Kit ) 的缩写。从字面意思翻译过来就是java开发的工具。它是一种用于构建在 Java 平台上发布的应用程序、applet 和组件的开发环境。
二、JDK的下载和安装
	推荐到甲骨文官网下载jdk软件包：https://www.oracle.com/
三、JDK环境变量配置
	cmd命令行输入java -version查看是否安装成功
		    java -verbose查看jdk安装路径
四、java程序的编译和运行
	新建文本文件，改后缀名为.java
	向java文件中输入如下代码：
	class Hello{
    		public static void main (String[] args){
        			System.out.println("Hello world!");
    		}
	}
	在cmd命令行窗口用javac Hello.java 编译生成.class文件
		            java Hello 运行生成的Java程序
